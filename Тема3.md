# Аналіз оперативної пам'яті: DDR4 vs DDR5

Цей документ містить порівняльний аналіз двох актуальних поколінь оперативної пам'яті (RAM) — DDR4 та DDR5, а також огляд конкретних моделей, доступних на ринку України (на прикладі магазину Rozetka).

## 1. Характеристики та порівняння стандартів

### Технічні відмінності
Оперативна пам'ять DDR5 (Double Data Rate 5) є еволюційним наступником DDR4, пропонуючи значні зміни в архітектурі для підвищення пропускної здатності та енергоефективності.

Основні зміни в DDR5:
* **Підвищена частота:** Стартова частота DDR5 (4800 МГц) значно вища за стандартні частоти DDR4 (3200 МГц).
* **Керування живленням (PMIC):** Контролер живлення перенесено з материнської плати безпосередньо на планку пам'яті, що покращує стабільність сигналу.
* **Архітектура каналів:** Один модуль DDR5 має два незалежних 32-бітних підканали, що підвищує ефективність доступу до даних.
* **ECC (On-die):** Вбудована корекція помилок на кристалі для підвищення надійності при високих швидкостях.

### Таблиця порівняння характеристик

| Характеристика | DDR4 | DDR5 |
| :--- | :--- | :--- |
| **Ефективна частота** | 2133 – 3200 МГц (до 5333+ з розгоном) | 4800 – 8400+ МГц |
| **Напруга живлення** | 1.2V | 1.1V |
| **Керування живленням** | На материнській платі | На модулі пам'яті (PMIC) |
| **Архітектура модуля** | Один 64-бітний канал | Два 32-бітних підканали |
| **Макс. об'єм чіпа (щільність)** | 16 Гбіт | 64 Гбіт |
| **Burst Length** | 8 | 16 |

## 3. Підсумкове порівняння обраних моделей

| Параметр | Kingston DDR4 (3200 МГц) | Kingston DDR5 (5200 МГц) | Висновок |
| :--- | :--- | :--- | :--- |
| **Пропускна здатність** | ~25.6 ГБ/с | ~41.6 ГБ/с | DDR5 має перевагу у **~60%** у швидкості передачі даних. |
| **Латентність (CL)** | CL16 (10 нс) | CL40 (15 нс) | DDR4 має меншу абсолютну затримку, що все ще важливо для деяких ігор. |
| **Ціна (орієнтовна)** | Нижча | Вища (~ в 2 рази за 1 ГБ) | DDR4 залишається бюджетним вибором, DDR5 — інвестиція в майбутнє. |
| **Сумісність** | Старіші платформи | Новітні платформи | Вибір залежить від материнської плати та процесора. |

**Загальний висновок:** Якщо ви збираєте бюджетний ПК, **DDR4** все ще залишається актуальним та вигідним рішенням. Однак для збірки високопродуктивної системи "з нуля" на базі нових процесорів доцільніше обирати **DDR5**, оскільки цей стандарт дозволить легше оновлювати систему в майбутньому.
